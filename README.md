### Dev

Install the dependencies

```sh
$ npm install -d
```

Dev

```sh
$ npm run dev
```

Build

```sh
$ webpack --config=dll.webpack.config.js // this will help improve the loading time
$ npm run build
```
Run

```sh
$ npm start
```
